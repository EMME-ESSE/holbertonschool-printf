## Dependencies

* The ````_printf```` function was coded on an Ubuntu 20.04 LTS machine with ````gcc```` version 4.8.4.
* The gcc code to compile is: ````gcc -Wall -Werror -Wextra -pedantic -std=gnu89 -Wno-format *.c````
